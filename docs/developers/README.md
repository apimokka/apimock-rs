# API mock (apimock-rs) Developers Documentation

## Getting started

Clone the repository and build:

```sh
git clone https://github.com/apimokka/apimock-rs.git
cd apimock-rs

cargo run
```

### Testing with codes

```
cargo test
```

## Build release binary

```bash
cargo build --release
```

Run the server:

```bash
target/release/apimock
```
